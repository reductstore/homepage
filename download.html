ReductStore supports Linux, macOs and Windows on AMD64 platform.

  • Linux
  • macOs
  • Windows

ReductStore

Binary

    wget https://github.com/reductstore/reductstore/releases/download/{{ site.projects.reductstore.version }}/reductstore.linux-amd64.tar.gz
    tar xfv reductstore.linux-amd64.tar.gz
    RS_DATA_PATH=./data ./reductstore

Docker

    docker run -p 8383:8383 -v ${PWD}/data:/data reductstore/reductstore:latest
